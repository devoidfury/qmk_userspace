# QMK Userspace

This is the QMK Userspace for the Bastard Keyboards keymaps.

You can read how to compile your own keymap on the official docs here: [https://docs.bastardkb.com/fw/compile-firmware.html](https://docs.bastardkb.com/fw/compile-firmware.html).


Maintained keyboards/layouts in this repo:

[charybdis/4x6/keymaps/devoid](./keyboards/bastardkb/charybdis/4x6/keymaps/devoid/readme.md)


```
qmk lint -kb bastardkb/charybdis/4x6 -km devoid
qmk compile -c -kb bastardkb/charybdis/4x6 -km devoid
```

# Dilemma Max (4x6) `devoid` keymap

The Dilemma Max (4x6) `devoid` keymap is inspired from the [Dilemma Max (4x6) vendor](../vendor) keymap, with my own customizations.

## Notable changes

### Danger layer
For less used tasks that are possibly disruptive. I kept accidentally tapping the EEPROM CLEAR / QMK BOOTLOADER keys so I moved them out of the way. This also seemed like a good place for RGB cycle keys, DPI cycle buttons, that sort of thing

Requires an intentional reach (hold 4/7 key).
